# Student Management

A simple web project for managing student information, similar to Moodle but smaller in scale.

## Project Description
- The project includes pages for **Login**, **Register**, and **Student Info**.
- Students can log in using their **name** and **password**.
- This project was created as part of a **Web Programming** course at the university.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP

## How to Use
1. Make sure you have a local server like **XAMPP** or **WAMP** installed.
2. Place the project folder inside the server's `htdocs` (or the appropriate local server folder).
3. Open your browser and go to `http://localhost/your_project_folder_name`.
4. Students need their **name** and **password** to log in.
